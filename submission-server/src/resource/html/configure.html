To make sure that Shenanigans only collects fingerprints from you (and not everyone else nearby), we'll instruct your phone to send out a WiFi probe request for a network named "shenanigans." You can do this by temporarily adding a new network called "shenanigans" to your device's list of preferred networks.

For best results, make sure that your device is not connected to a WiFi network. If the phone is connected, it doesn't send out the full WiFi fingerprint.

Instructions for iPhone / iPad

  1. Go to Settings->WiFi. Make sure that WiFi is enabled.
  2. (optional) If your device is presently connected to a WiFi network, disconnect from that network. If you skip this step, Shenanigans can only get a partial fingerprint that won't include your full list of preferred networks.
  3. Press "Other..." at the bottom of the list of networks.
  4. Enter shenanigans as the network name.
  5. Tap Join.
  6. You'll see an error message saying Could not find the network "shenanigans" -- that's ok, the network doesn't really exist, we just told your phone to send out a network probe request. Click dismiss.

Instructions for Android devices

  1. Make sure that WiFi is enabled on your device.
  2. (optional) If your device is presently connected to a WiFi network, tap on that network and then tap "Forget." You'll have to re-add this network later. If you skip this step, Shenanigans can only get a partial fingerprint that won't include your full list of preferred networks.
  3. Add a new WiFi network named "shenanigans" to your device. This confirms that it's your device, and not your neighbor's. Use "none" for the security mode.
  4. Tap "Save."
  5. Find "shenanigans" in your network list, tap on it, and tap "Forget."
